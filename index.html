Flag

Quiz Rules

  • There are 4 rounds with 5 questions in each round.

  • Each round represents a geographic grouping of flags.

  • Flag questions are multiple choice. Choose 1 correct answer from a list of 4 options.

  • Each correct flag answer earns you 2 points.

  • There is a bonus question at the end of each round. This is a true or false question and is worth 4 points.
